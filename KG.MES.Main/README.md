# XML Order Reader - Blazor Web Application

Веб-приложение для чтения XML файлов заказов и экспорта данных в 1С.

## Функциональность

- 📄 Загрузка XML файлов заказов через веб-интерфейс
- 👀 Просмотр деталей заказов в удобном формате
- 📊 Отображение товаров и материалов
- 🔄 Экспорт данных в 1С через API
- 📱 Адаптивный дизайн для всех устройств

## Технологии

- ASP.NET Core 8.0
- Blazor Server
- Bootstrap 5.3
- Bootstrap Icons
- System.Xml.Serialization

## Структура проекта
KG.MES.Main/

├── Components/ # Blazor компоненты

## Установка и запуск

1. Клонируйте репозиторий
2. Восстановите зависимости: `dotnet restore`
3. Запустите приложение: `dotnet run`
4. Откройте в браузере: `https://localhost:7000`

## Настройка

Настройки подключения к 1С API находятся в `appsettings.json`:

```json
{
  "1CApi": {
    "BaseUrl": "http://localhost:5000/api"
  }
}

## Использование

Нажмите "Загрузить XML" для выбора файла

Просмотрите детали заказа во вкладках

Используйте "Экспорт в 1С" для отправки данных
